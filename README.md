# Emotune

Emotion-Based Music Recommendation System powered by Face, Voice and Text Emotion Detection.

## Features

- Face Emotion Recognition using DeepFace
- Voice Emotion Detection using CNN
- Text Sentiment Analysis
- Mood Analytics Dashboard
- Mood History Tracking
- Personalized Music Recommendations

## Tech Stack

- Python
- Flask
- TensorFlow
- DeepFace
- SQLite
- HTML
- CSS
- JavaScript

## How To Run

1. Install requirements

pip install -r requirements.txt

2. Create database

python create_db.py

3. Start application

python app.py

4. Open browser

http://127.0.0.1:5000
